# Construct Recipes

Constructs are cross-cutting building blocks — storage and behavior patterns
an app composes into its own features, rather than mountable components.
These recipes are short implementation entry points; each points at the
solution doc that owns the WHY.

## Recipes

| Recipe | Use when |
| --- | --- |
| [App User Settings](user-settings-README.md) | Your app needs durable per-user choices (toggles, picks, standing policies) applied on every turn. |
